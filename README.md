# LunaSQL
基本功能：<br>
①数据库管理工具的基本功能（增删改查，表结构，索引)<br>
②数据库日志/参数的图形化界面管理<br>
③自定义测试用例测试评估数据库性能，记录结果。<br>
④soar的sql优化，重写等常用功能的图像化界面实现<br>
(https://github.com/XiaoMi/soar)

<hr>
使用需要配置JRE（java版本8以上)<br>
20.3.12<br>
